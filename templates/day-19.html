flash-сообщения

Функция flash() в Flask — это эффективный способ отображения временных сообщений пользователю. Она может использоваться для отображения различных сообщений, включая сообщения об ошибках, уведомления, предупреждения и сообщения о статусе.

Они выглядят так:

flash(message,category)

Аргументы: message - сообщение, которое нужно вывести. category - категория сообщения, можно использовать любую строку. Рекомендуются следующие значения: 'message' для любого типа сообщения, 'error' для ошибок, 'info' для информационных сообщений 'warning' для предупреждений.

Чтобы извлечь флэш-сообщение из сессии, где оно хранится, и отобразить его на шаблоне, мы используем функцию get_flashed_messages(). 

 get_flashed_messages(with_categories, category_filter)

with_categories : Необязательный кортеж параметров для упоминания категории (ошибка/информация/предупреждение) category_filter : Необязательный параметр для фильтрации и отображения только определенных сообщений.
