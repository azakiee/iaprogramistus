Авторизация и сессии

На сайт должен кто то зайти). Система поможет нам с этим: 'username' сообщает нам имя пользователя, 'password', соответственно, пароль.

        users = [
            {'username': "Имя пользователя", 'password': "Пароль пользователя"}
        ]

Но так же система должна запомнить пользователя. В этом ему поможет - "sessions", с помощью которого пользователь не будет несколько раз регестрироваться. Для этого при открытии пути к главной странице создаем переменную, в которую закладываем класс sessions с атрибутом get. В скобках прописываем, что мы запоминаем имя:

    @app.route('/start')
    def start():
        username = session.get('username', None)
        return render_template('start.html')
